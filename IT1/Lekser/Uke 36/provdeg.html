5.1 Prøv Selv

  1. Hvor mange timer bruker du daglig på Internett?
    Jeg vil tro jeg bruker ca. 6-7 timer på Internett hverdag.

  2. Hvilke apper og sosiale medier bruker du daglig?
    Jeg bruker Facebook, Instagram, Twitter, VSCO, Netflix, Spotify, Tumblr, Ruter Reise, YouTube, Instagram og Snapchat daglig.

  3. Forsøk å finne ut hvor mange i Norge i dag som bruker Facebook og Snapchat daglig.
    Omtrent tre millioner Nordmenn bruker Facebook daglig og ca. to millioner Nordmenn bruker Snapchat daglig.

  4. Forsøk å finne ut hvor mange i Norge som ikke bruker internett.
    Kun 4% av Norges befolkning bruker ikke internett.

  5. Hvordan kan økningen i ensomhet som Ungdata-undersøkelsen viser skyldes den digitale måten vi kommuniserer på?
    Ungdommer bruker mer tid på sosiale medier og Internett, og mindre tid sammen med venner ansikt-til-ansikt.Vi bruker mer tid på Internett enn med mennesker og blir derfor ikke like godt kjent og mange kan føle seg alene.

  6. Hvilke fire punkter kjennetegner kommunikasjon på sosiale medier?
    • Varig largring: Samtaler og kommunikasjonen blir lagret på nettet i lang tid.
    • Søking: Digitale identiteter er søkbare.
    • Gjenbruk: Det kan være vanskelig å skille orginalen fra kopien når vi bruker informsjon fra et annet sted.
    • Usynlige tilskuere: Lite kontroll over hvem som følger med på hva vi gjør.


5.2 Prøv Selv

  1. Hva er forskjellen på vertikale og horisontale sosiale systemer?
    Vertikale sosiale systemer er vi fysisk sammen i større grupper, som på skolen, trening og hos familie eller venner. Horisontale sosiale systemer er online eller på sosiale medier, som Facebook, Instagram, Twitter, YouTube og blogger.

  2. Hvordan kan teknologi brukes til å styre kommunikasjon og sosial tilhørighet?
    Det er lettere å komme i kontakt med andre gjennom sosiale medier. Det tar lang tid å dra bort til en venn og ringe på for å spørre om noe, mens på sosiale medier tar det bare noen sekunder. Det kan være lettere å bli invitert til arragementer på nettet, som kan styre sosial tilhørighet.

  3. Hvordan kan språklige og kulturelle minoriteter bruke sosiale medier?
    De kan snakke med folk som ikke er i lokal miljøet. Det er lettere for dem å komme i kontakt med andre som snakker samme språk eller har samme kultur som deg. Det er også lettere å søke opp normer og tradisjoner for å komme lettere med i samfunnet.

  4. Kartlegg din egen hverdag. Hvor mange timer bruker du på skole, trening, henge med venner, familie, alene på rommet og timer til å sove? Hvor brukte du sosiale medier mest?
    Aktivitet Timer
    Skole 7 timer
    Trening 1,5 time
    Sove 7 timer
    Alene på rommet 4 timer
    Henge med venner 1 time
    Jeg brukte sosiale medier mest når jeg er alene på rommet.

  5. Hvordan kan sosiale medier misbrukes av religiøse og politiske ekstremister?
    Sosiale medier kan misbrukes ved at de legger ut forstyrrende videoer og falske fakta. De prøver da å få andre til å bli enige med dem selv om det de sier ikke nødvendig vis er sant. De kan også på til angrep på mot folk de ikke er enige med. Det kan være farlig fordi det kan nå mange og kan påvirke meningene til andre på en negativ måte.
